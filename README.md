## About

**The Civic Router: Internet for All** is a 6-chapter interactive flippable comic about community networks, digital inclusion, internet governance, cybersecurity, and digital democracy. It follows Amina, Tunde, Mama Grace, Mr. Byte, and the Disconnect Monster through the story of a rural Nigerian community that fights the digital divide — and wins.

Written by Gbemisola Esho, Google Developer Expert (GDE) and Women Techmakers (WTM) Ambassador. Produced in association with AFRALO / ICANN At-Large.

## 🌐 Live Demo

👉 **[https://apinke.github.io/civic-router/](https://apinke.github.io/civic-router/)**

Features:
- 🔄 Flip each page to reveal key lessons and discussion points
- 🔊 **Text-to-Speech** — auto-reads each page aloud, front and back
- 👩 / 👨 Female and male voice options with speed control
- ← → Navigate with buttons, arrow keys, or swipe
- 📋 Table of Contents — jump to any comic
- Works on mobile, tablet, and desktop

## 🚀 How to Use

### Read Online
Visit: **https://apinke.github.io/civic-router/**

### Fork and Translate
1. Fork this repo
2. Edit the text content in `index.html`
3. Translate into Hausa, Yoruba, Igbo, Swahili, French, or any language
4. Enable GitHub Pages to share your translated version

### Download and Use Offline
Download `index.html` — works fully in any browser without internet.
Perfect for community training sessions and workshops.
